    HTML/CSS

    If you can imagine a time before the invention of the Internet, websites didn’t exist and books printed on paper tightly bounded, were the primary source of information. It took a considerable amount of effort and reading to track down the exact piece of information you were after.


    Today you can open a web browser, jump over to your search engine of choice, and search away. Any bit of imaginable information rests at your fingertips. And chances are someone somewhere has built a website with your exact search in mind.


    So, what is HTML and CSS?

    HTML(Hypertext Markup Language) is the visual language of the webpage and every web browser understands it. As a web developer, it is important to know at least the Basics of HTML as it the central point of the contents on the web page.


    CSS, or Cascading Style Sheets, is a presentation language created to style the appearance of web contents created by the HTML using fonts, colors, margin, height, width etc.. It works alongside with HTML as it gives it a good look.

    JAVASCRIPT

    Javascript is the most used programming language all over the world. Like HTML and CSS, every web browser understands Javascript and every web browser has a Javascript Engine in them. Example: Chrome has V8 as its Javascript Engine.


    A better way to also understand what JavaScript does is to think about certain web features you use every day and usually take for granted, like when your Facebook timeline automatically updates on your screen or Google suggests search terms based on a few letters you’ve started typing. In both cases, that’s JavaScript in action


    If you’re hoping to break into a career in tech, your question might sound more like “What is JavaScript and do I need it?”. If you’re interested in web development, the answer is a resounding YES.

    What can JavaScript do?

    • Mobile Application Development: That's it! Young people love games like I do, so if you want to learn how it was built then Javascript might be good a option for you to pick.
    • Web Pages Interactivity: You want your page look more than just static texts? Javascript is an answer to the question that bothers you about adding something more catchy.
    • Creating Web Games: Ever wondered how or played a game online on your browser? Javascript probably helped work that out.
